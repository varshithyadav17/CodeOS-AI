# CodeOS AI

CodeOS AI is an AI-powered software engineering workspace designed to help developers understand, explore, and analyze large codebases. It combines repository intelligence, architecture visualization, knowledge graphs, AI-assisted documentation, and conversational code understanding into a unified interface.

## Features

- Repository analysis and exploration
- AI-powered code assistant
- Interactive Knowledge Graph
- Architecture visualization
- Engineering Memory
- Timeline analysis
- AI-generated documentation
- Automated code reviews

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Flow

### Backend
- FastAPI
- MongoDB

### AI
- Google Gemini

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
python -m venv .venv

# Windows
.venv\Scripts\activate

pip install -r requirements.txt
uvicorn server:app --reload
```
