# CodeOS AI

CodeOS AI is an AI-powered software engineering workspace that helps developers analyze repositories through AI-assisted code understanding, architecture visualization, engineering memory, knowledge graphs, documentation generation, and repository intelligence.

## Tech Stack

- React
- Vite
- Tailwind CSS
- FastAPI
- MongoDB
- Google Gemini
- React Flow

## Features

- AI Chat
- Knowledge Graph
- Architecture Analysis
- Engineering Memory
- Timeline Analysis
- Documentation Generation
- AI Code Reviews

> This project is actively under development.
